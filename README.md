# 3D Portfolio Model

## Description

Simple 3D portfolio model that allows you to sign your data together
with your experience ind nice way to apply for suitable matching jobs.

## Techiques Used

- HTML
- CSS
- Javascript
- vite
- React

  ## Sections

  - Intro and Landing 3d Model.
  - Overview.
  - Work Experience.
  - Project Samples.
  - Testimonials.
  - Contact Form.
 
    ## How to run
   ###  From the commandline run
    - npm i
    - npm run dev
